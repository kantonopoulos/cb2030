# CB2030 Computer Lab 4: Proteogenomics Analysis

## Instructions

### 0. Preparation (Optional but Recommended)
Before starting, it is recommended to read the following papers for background knowledge and fully understanding the dataset and the study itself:
- [Whole-genome sequence association analysis of blood proteins in a longitudinal wellness cohort](https://genomemedicine.biomedcentral.com/articles/10.1186/s13073-020-00755-0)
- [Quantitative variability of 342 plasma proteins in a human twin population](https://pubmed.ncbi.nlm.nih.gov/25652787/)

### 1. Download and Setup
1. Download the repository and unzip it.
2. Don't forget to unzip also the file `data/syn.Genotype_annotation.txt.gz`.

### 2. Run the Notebook
1. Open the notebook `CB2030_lab4.ipynb` in your preferred Jupyter environment.
2. Run the notebook step by step.

### 3. Answer the Questions
- Answer the questions directly in the notebook.
- Some questions require minor coding, while others involve running the provided code blocks to generate plots.

### 4. Submit Your Report
1. Write your answers and code in the notebook.
2. Export the completed notebook as a PDF.
3. Submit the PDF as your report. Ensure the file is named properly to include your name(s).
